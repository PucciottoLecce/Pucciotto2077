# Pucciotto Heist 2077 🥙

Mini-gioco arcade 3D in stile Donkey Kong. Tre mercenari rubano i premi alla **PUCCIOTTO CORP**.

## Giocare subito
Apri `index.html` in un browser moderno (serve internet per Three.js la prima volta).

## Pubblicare sul web (consigliato per smartphone)
Il gioco è una **PWA**: se lo metti online con HTTPS, da smartphone si può "installare"
sulla home (Aggiungi a schermata Home) e funziona anche offline. Opzioni gratuite:
1. **GitHub Pages** — crea un repo, carica questa cartella, attiva Pages nelle impostazioni.
2. **Netlify Drop** (app.netlify.com/drop) — trascina la cartella, hai subito un link.
3. **itch.io** — carica lo zip come gioco HTML5.

## Comandi
- ◀ ▶ / A D — muoviti · ⬆ / W — scala (se non c'è una scala vicino: salta)
- SPAZIO — salta (+100 se scavalchi un ostacolo) · E — abilità
- Su smartphone compaiono automaticamente i pulsanti touch.

## Personaggi
- **Cyber Ninja** — veloce, salto alto. [E] Scatto nella direzione in cui guarda.
- **Bounty Hunter** — lento ma regge un colpo extra. [E] Scudo 2s.
- **Street Hacker** — equilibrato. [E] Hack: distrugge l'ostacolo più vicino (+150).

## I 5 livelli e i premi
1. Metro abbandonata → 🔑 Portachiavi
2. Tetto sotto la pioggia → 🥔 Crocchettone
3. Metro zona profonda (treni merci della Corp!) → 🥙 Pucciotto
4. Scalata della Torre Corp (vento + laser) → 👕 Maglietta
5. Attico col boss Chimera → 👕🥙 Maglietta + Pucciotto
